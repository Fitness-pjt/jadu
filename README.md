# PROJECT

`프로젝트 개요`

### | 목표

- 웹 아키텍처를 이해하고 이를 활용하여 웹 프로젝트를 설계하고 구현할 수 있다.
- Spring Boot와 Vue를 이해하고 활용할 수 있다.
- REST API와 Vue Router을 활용하여 SPA 방식으로 프로젝트를 구성할 수 있다.
- 기존 관통 프로젝트에 새로운 아이디어를 추가하여 과정에서 배운 기술들을 다양하게 구현해 본다.
